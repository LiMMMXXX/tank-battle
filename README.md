# 🎮 坦克大战 (Tank Battle)

一个经典的HTML坦克大战游戏，纯前端实现，无需任何依赖。

## 🕹️ 游戏玩法

- **移动**: W/S/A/D 或方向键 ↑↓←→
- **射击**: 空格键
- **暂停**: P键

## 🎯 游戏目标

保护你的基地🦅，消灭所有敌方坦克！

## ✨ 特性

- 经典坦克大战玩法
- 多种地图地形（砖墙🧱、钢铁🛡️、水域💧）
- **5种道具系统**:
  - 🛡️ 护盾 - 抵挡一次攻击
  - ⚡ 速射 - 短时间内快速射击
  - 💥 大炮弹 - 可摧毁钢铁墙壁
  - ❤️ 加命 - 增加一条生命
  - ❄️ 冻结 - 冻结所有敌人
- 难度递增的关卡系统
- 爆炸粒子特效
- 敌方AI（寻路+瞄准射击）
- 子弹碰撞系统
- 暂停功能

## 🚀 运行方式

直接在浏览器中打开 `index.html` 即可开始游戏！

或者使用任意HTTP服务器：
```bash
python -m http.server 8000
# 访问 http://localhost:8000
```

## 🛠️ 技术栈

- HTML5 Canvas
- Vanilla JavaScript (ES6+)
- 纯CSS动画
- 分离式文件结构

## 📁 文件结构

```
tank-battle/
├── index.html    # HTML 骨架
├── style.css     # 所有样式
├── game.js       # 游戏逻辑
└── README.md
```

## ⚡ 优化特性

- **Delta Time 游戏循环** — 移动/射击/计时器全部基于真实时间，60Hz/144Hz/240Hz 体验一致
- **状态驱动输入** — keydown/keyup 跟踪按键状态，消除 OS 按键重复延迟
- **模块化代码** — HTML/CSS/JS 分离，便于维护
